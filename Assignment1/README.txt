Compile instructions:

Java program takes input file name and output file name as command
line arguments.

e.g.
	javac FCFS.java
	java FCFS inputFCFS.txt outputFCFS.txt

C program prompts you for the number of processes and file names.

	e.g. 
	gcc -o FCFS FCFS.c
	./FCFS

	3
	inputFCFS.txt
	outputFCFS.txt
